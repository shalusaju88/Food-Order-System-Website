# Delizioso - Premium Food Order System

A stunning, highly-responsive, and modern front-end application for a premium food ordering service. Built entirely with Vanilla HTML, CSS, and modern JavaScript, prioritizing rich aesthetics and dynamic user experience.

## ✨ Features
- **Striking Visual Design**: Glassmorphism effects, a sophisticated dark-mode palette, and elegant typography (Playfair Display & Outfit).
- **Interactive Menu**: Dynamic card hovers, 3D transformations, and micro-animations to create a luxurious feel.
- **Product Details Modal**: Fullscreen-capable product preview showing high-res imagery, mock metadata (calories, prep time), and quantity selectors.
- **Fully Functional Cart**: Slide-out shopping cart sidebar with real-time updates for subtotal, dynamic delivery fees, and total pricing.
- **Mobile Optimized**: Custom hamburger navigation with an off-canvas menu and adaptive typography ensuring maximum screen real estate on mobile devices.
- **AI-Generated Assets**: Beautiful, high-resolution food photography assets tailored locally for the project.

## 🚀 Tech Stack
- **HTML5**: Semantic structure
- **CSS3 (Vanilla)**: CSS Grid, Flexbox, Custom Properties (Variables), advanced `@keyframes` animations, and glassmorphic UI tricks
- **JavaScript (Vanilla)**: Custom DOM manipulation and state tracking
